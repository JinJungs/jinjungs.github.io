### 백준 4673 - 셀프 넘버


```python
a = [i for i in range(1,10000)]

def d(n):
    return sum(map(int,str(n))) + n

for j in range(1,10000):
    if d(j) in a:
        if d(j)>10000:
            break
        a.remove(d(j))
    
print('\n'.join(map(str,a)))
```
