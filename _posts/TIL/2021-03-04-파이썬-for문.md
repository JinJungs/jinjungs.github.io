### [파이썬] for문

- 개인적으로 range 함수를 제일 많이 사용하게 되는 것 같다. 다음은 1부터 10까지 더하는 로직을 구현한 것이다.

  ```python
  add = 0
  for i in range(1,11):
  	add = add + i
  
  print(add)
  ```
