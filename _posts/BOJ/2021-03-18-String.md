### BOJ 10809 - 알파벳 찾기


```python
x = list(map(str,input()))
a = [-1 for i in range(26)]
# 각 인덱스 위치는 ASCII 코드 -65
for j in range(len(x)):
    index = ord(x[j])-97
    if a[index] == -1:
        a[index] = j

print(" ".join(map(str,a)))
```

### BOJ 2675 - 문자열 반복


```python
n = int(input())
for i in range(n):
    r, x = input().split(' ')
    a = list(map(str,x))
    p = []
    for j in range(len(a)):
        for _ in range(int(r)):
            p.append(a[j])
    print("".join(p))
```

    3 ABC
    AAABBBCCC
    5 /HTP
